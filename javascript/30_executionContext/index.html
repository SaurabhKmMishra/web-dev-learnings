What is an execution context in JavaScript, and what are the different types of execution contexts?

:> An execution context in JavaScript is an environment in which JavaScript code is evaluated and executed. It consists of variables, functions, and the this keyword that are accessible at a particular moment in the program.

:> When JavaScript code runs, the JavaScript Engine creates an execution context to keep track of the currently executing code.

=> it has following two types:-

1:> GLOBAL EXECUTION CONTEXT(GEC) :> Created when the script first runs. > Represents the global scope (window in browsers, global in Node.js). > Only one global execution context exists. > Variables and functions declared outside any function belong to this context.

2:> Function Execution Context (FEC) Created when a function is called. Each function has its own execution context.

Contains: Function scope variables (arguments object, local variables).
